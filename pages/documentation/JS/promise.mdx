# Promise

## 1. 实现 Promise.all

```js
Promise.all = function (values) {
  return new Promise((resolve, reject) => {
    let idx = 0;
    let result = [];
    values.forEach((item, index) => {
      Promise.resolve(item).then((value) => {
        result[index] = value;
        if (++idx === values.length) {
          resolve(result);
        }
      }, reject);
    });
  });
};
```
